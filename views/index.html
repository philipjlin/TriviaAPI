Trivia API - REST Implementation


https://triviaapi.onrender.com/randomQuestion/:questionLevel

Route Action
GET Gets random question of specified level (Blue, Green, Yellow, Red, Purple)

https://triviaapi.onrender.com/questions

Route Action
GET Retrieves all questions in database
POST Creates a new question to add to database

https://triviaapi.onrender.com/questions/:questionId

Route Action
GET Retrieves question with specified object id
PATCH Updates question with specified object id
DELETE Deletes question with specified object id
